# Flutter-app

## Backend Setup

1. Navigate to the `backend` directory.

    ```bash
    cd backend
    ```

2. Open `db.js` and update your SQL connection details.

3. Install dependencies using npm.

    ```bash
    npm install
    ```

Now, your backend is set up with the updated SQL connection details.
